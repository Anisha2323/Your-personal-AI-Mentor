# Your-personal-AI-Mentor
Your Personal Mentor is an AI-powered Chrome extension built to support learners while they solve Data Structures &amp; Algorithms (DSA) problems on platforms like LeetCode.
It doesn’t give away answers—instead, it behaves like a real mentor sitting beside you. 
It offers gentle hints, small nudges, and structured guidance to help you think through the problem step-by-step so you can arrive at the solution on your own.
